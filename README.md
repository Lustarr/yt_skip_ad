# YT Skip Ad(MV3)

## 安裝
1. 下載或 clone 專案  
   ```bash
   git clone https://github.com/yourname/yt-skip-ad-mv3.git
   ```
2. 開啟 Chrome 擴充管理頁(chrome://extensions/)
3. 開啟右上角「開發人員模式」
4. 點擊「載入未封裝項目」，並選擇此專案資料夾
5. 開啟 YouTube 頁面即可自動生效，若第一次安裝後未立即作用，請重新整理頁面一次

## 權限說明
- scripting: 在網頁執行 script
- activeTab: 取得分頁與網址控制權限
- debugger: 允許 DevTools Protocol
